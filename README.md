## Project 'Ambito'

Tutorial guide for setting up a NextAuth.js for user authentication with email and password with the following requirements:
 - Node.js v20
 - MongoDB Community Edition 7.0
 
## Steps
Run the following commands:-

#### 1. Git clone

#### 2. npm i

#### 3. npm run dev
